#Debt Collector Game

Unity oraz programu do modelowania 3D Blender. Zadaniem gracza jest ze-
branie wymaganej ilości długu w wyznaczonym czasie. Rozgrywka ta oferuje
ćwiczenie pamięci gracza poprzez zapamiętywanie przedmiotów
